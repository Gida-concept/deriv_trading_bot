HOW TO GET YOUR BINANCE API KEYS
  1. Log into your Binance account at binance.com
  2. Go to API Management in your account settings
  3. Create a new API key and label it (e.g., "Trading Bot")
  4. Enable Futures and Reading permissions
  5. Copy both the API Key and Secret Key
  6. Paste them into the form above and click "Save Credentials"
